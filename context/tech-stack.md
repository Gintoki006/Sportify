# Tech Stack

## Overview

Sportify will use a modern web stack optimized for rapid development, secure authentication, and rich data visualization.

## Frontend

- Framework: Next.js
- Library: React.js
- Language: JavaScript
- Styling: Tailwind CSS
- Charts: Recharts
- Authentication: Clerk

## Backend

- Runtime: Node.js
- Framework: Express.js
- ORM: Prisma

## Database

- PostgreSQL (hosted on Supabase)

## Notes

- Recharts will power performance dashboards and progress views.
- Clerk handles auth flows (sign up, login, password reset).
